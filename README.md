# This Is What Happened

A living family museum and archive.

## Main files
- `index.html` — Entrance Hall
- `family-tree.html` — interactive four-family tree
- `family.html` — family collection page
- `person.html` — person exhibit page
- `timeline.html` — family timeline
- `add-a-discovery.html` — simple contribution form
- `css/museum.css` — shared visual design
- `js/museum.js` — shared people data and interactions
- `images/` — museum images

GitHub Pages should publish from the `main` branch and the repository root.
